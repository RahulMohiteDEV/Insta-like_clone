# 📸 Instagram Clone

A **Full Stack Instagram Clone** built using the **MERN Stack (MongoDB, Express.js, React.js, Node.js)**.  
This application allows users to **register, login, upload posts, like posts, and view feeds**, similar to Instagram.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 👤 User Registration & Login
- 📸 Upload Posts with Images
- ❤️ Like / Unlike Posts
- 📰 View Posts Feed
- 🧾 User Profile
- 🍪 Secure Authentication using Cookies
- 🔒 Password Hashing with Bcrypt
- 🌐 REST API using Express
